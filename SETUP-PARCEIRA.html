Os 3 essenciais — pra app começar a funcionar

S
Supabase — banco de dados + login
Onde ficam os usuários, histórias e pagamentos
Essencial ▾
Pra que serve: é o "cérebro" do app. Guarda quem está cadastrada, quais histórias existem, quem assinou, quem favoritou o quê. Grátis até 50 mil usuárias por mês.

Abrir Supabase →

Opção A — fazer eu mesma

  1. Clica em "Start your project" e faz login com GitHub (mais fácil) ou cria conta com e-mail.
  2. Clica em "New Project".
  3. Nome do projeto: alluretv-prod
  4. Senha do banco: deixa o Supabase gerar uma forte e SALVA num bloco de notas. Você vai precisar.
  5. Região: East US (North Virginia) — é a mais perto das nossas usuárias.
  6. Clica em "Create new project" e espera ~2 minutos pra ele criar.
  7. Quando abrir, no menu da esquerda clica em "SQL Editor". Vou te mandar um arquivo chamado 0001_init.sql — abre ele num bloco de notas, copia tudo, cola no SQL Editor e clica em "Run" (ou Ctrl+Enter). Vai aparecer "Success. No rows returned".
  8. Volta pro menu, clica na engrenagem (Project Settings) → "API".
  9. Copia esses 3 valores e me manda:
    • Project URL (algo como https://abc123xyz.supabase.co)
    • Chave anon public (texto enorme começando com eyJ...)
    • Chave service_role secret ⚠️ essa é tipo senha de banco, não compartilha com mais ninguém

Opção B — me adicionar como cocriador

  1. Faz tudo até o passo 7 acima (criar projeto + rodar o SQL).
  2. No menu da esquerda, clica na engrenagem → "Team".
  3. Clica em "Invite a member", digita lucasburque93@gmail.com, escolhe role "Owner".
  4. Pronto, eu pego as chaves direto.
$
Stripe Brasil — onde a gente recebe os pagamentos
$14,99/mês das assinantes cai aqui (CNPJ seu)
Essencial ▾
Pra que serve: é onde processamos cartão de crédito, Apple Pay e Google Pay. O dinheiro cai na sua conta bancária pelo CNPJ que você já tem. Stripe tira uma porcentagem (~3,5%) por transação.
Importante: Stripe Brasil pede CNPJ. Como você já tem, é tranquilo. Eles aprovam em ~24h. Enquanto não aprova, podemos usar em modo TEST (com cartões falsos pra a gente testar).

Abrir Stripe →

Opção A — fazer eu mesma

  1. Cria conta em dashboard.stripe.com com seu e-mail. Escolhe país Brasil.
  2. Preenche os dados do CNPJ (vai pedir documento, conta bancária — dá pra completar depois). Aprovação leva até 24h.
  3. Enquanto isso, no canto superior esquerdo certifica que tá no modo "Test mode" (botãozinho ligado).
  4. No menu, vai em "Products" → "Add product".
  5. Nome: AllureTV Monthly. Preço: 14.99 USD, recorrência Monthly. Save.
  6. Vai abrir uma página com o produto. Copia o Price ID (algo como price_1AbcDef...) e me manda.
  7. Vai em "Developers" → "API keys". Copia:
    • Publishable key (começa com pk_test_...)
    • Secret key (começa com sk_test_...) — clica em "Reveal" pra mostrar
  8. O Webhook Secret a gente configura depois quando o app tiver domínio. Por agora, ignore.

B
Bunny Stream — onde os vídeos vão ficar
$5/mês — seus vídeos hospedados rapidíssimo no mundo todo
Essencial ▾
Pra que serve: guarda os vídeos das histórias e entrega rápido em qualquer celular do mundo. É como YouTube, mas pago — sem anúncios, sem algoritmo, e a gente tem controle total. $5/mês fixo, paga em dólar com cartão internacional.

Abrir Bunny.net →

Opção A — fazer eu mesma

  1. Cria conta em bunny.net com e-mail. Adiciona um cartão (eles cobram $5/mês depois do primeiro vídeo).
  2. No menu superior clica em "Stream" → "Add Library".
  3. Nome: alluretv. Região principal: New York. Save.
  4. Clica na biblioteca recém-criada pra abrir.
  5. Lá em cima vai ter o Library ID (um número grande tipo 123456) — anota.
  6. Clica em "API" (menu da biblioteca). Copia a Library API Key (texto longo).
  7. Volta pra biblioteca. No menu lateral clica em "Pull Zone". Anota o hostname que aparece (algo como alluretv.b-cdn.net).
  8. Ainda no Pull Zone, clica em "Token Authentication" → ativa a opção "Require token authentication". Copia o Authentication Key (texto longo).
  9. Me manda esses 4 valores: Library ID, API Key, hostname, Authentication Key.

  2. No menu superior, clica no seu avatar → "Account Settings" → "Team".
  3. Adiciona lucasburque93@gmail.com como membro com permissão de "Stream".
  4. Pronto.

    Stream → Library → número no topo

    Stream → Library → API → Library API Key

    Stream → Library → Pull Zone hostname
  • Bunny: BUNNY_SECURITY_KEY
